spring boot起步——2.返回json数据

发表于 2017-08-18 | 分类于 技术 ， 框架 ， spring boot

spring Boot引用了json解析包Jackson，rest接口中可以通过return语句直接响应给浏览器json数据，我们可以在自己的实体类中提供json属性的注解，对时间进行格式化，对一些字段进行忽略等。

返回json的步骤：

  1. 编写实体类Person
  2. 编写restController：HelloController
  3. 在HelloController中加上@RestController和@RequestMapping注解

浏览器的显示的json字符串：{“id”:1,”name”:”list”,”realName”:”李斯特”,”birthDay”:”2017-08-17 06:28:58”}
