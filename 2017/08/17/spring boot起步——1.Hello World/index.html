spring boot起步——1.Hello World

发表于 2017-08-17 | 分类于 技术 ， 框架 ， spring boot

步骤：

  1. 在eclipse中新建maven Java工程
  2. 在pom.xml中添加Spring BootMaven的依赖
  3. 编写启动类
  4. 运行程序

新建maven项目

  1. 在eclipse文件视图中点击右键New -> Other,找到maven选择Maven Project
  2. 下一步后选择org.apache.maven.archetypes —— maven-archetype-quickstart
  3. 填写GroupID和artifactID，点击finish完成创建

    修改pom.xml

    1.在pom.xml文件中引入spring-boot-start-parent，可以提供dependency management 依赖管理，引入后其他dependency不需要version

2.因为是web工程，所以需要在pom.xml中引入spring-boot-starter-web,其中包含了spring webmvc和Tomcat等web开发的特性

3.如果需要使用Main启动spring，plugin就必须添加(不添加也可以运行)

spring boot 建议将main方法配置在包报名下

编写启动类

@RestController:返回json字符串数据，直接可以编写restFull接口,相当于同时增加了@Controller和@ResponseBody
@SpringBootApplication申明让spring boot自动给程序进行配置，等价于使用@Configuration，@EnableAutoConfiguration和@ComponentScan

运行

右键Run As -> Java Application

控制台看到 Started App in 3.444 seconds (JVM running for 4.021) 启动成功

打开浏览器输入http://127.0.0.1:8080,就可以看到hello world

完整的pom.xml
